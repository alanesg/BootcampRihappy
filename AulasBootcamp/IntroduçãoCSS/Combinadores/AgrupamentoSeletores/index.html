Agrupamento de Seletores

Podemos aplicar mesmas regras CSS para vários seletores diferentes. Para isso, podemos separar os seletores utilizando uma vírgula(,).

Assim, não precisamos ficar copiando e colando os mesmos códigos para todos, nós podemos fazer um só bloco de CSS e reutilizas as mesmas regras!
